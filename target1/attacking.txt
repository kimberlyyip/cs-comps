Step 3. See apache server and SSH ports are open 
Step 4. Connect to the apache server 
Step 5. Try cole and Robbie like file names, find secrets.txt and /web/secrets.txt
Step 6. From web add /.ssh into the url gain RSA key 
Step 7. Download the ssh private and public keys and the RSA key to your attacking machine
